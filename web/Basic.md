
CSS position

- 문서 상에서 요소를 배치하는 방법을 지정한다. 

relatvie: 원래 내가 있어야 하는 위치에서 움직이는 것

absolute: 부모를 기준으로 절대적으로 움직이게 됨

fixed: 무조건 고정값
